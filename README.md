# Traffic-Light
An simple Traffic light using LED and Arduino 

# Component used
1. Arduino UNO
2. Resistor (1kw) 
3. Led's
4. Wires

# Connections 
1. Red LED: Positive (Anode) pin connected to Digital Pin 11 via a 220Ω Resistor    
2. Yellow LED: Positive (Anode) pin connected to Digital Pin 10 via a 220Ω Resistor      
3. Green LED: Positive (Anode) pin connected to Digital Pin 9 via a 220Ω Resistor       
4. Common Ground: Saari LEDs ki Negative (Cathode) pins Arduino ke GND pin se connected hain
